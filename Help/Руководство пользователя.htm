Знакомство с программой

 

После запуска приложения Вы находитесь в главном окне управления.

 

 

Доступные модули управление находятся в меню Файл. Выберите пункт меню Файл и необходимый Вам модуль для начала работы.

 

Для всех модулей доступен единый функционал: добавление, удаление, сохранение, отмена, обновление и экспорт данных, а также закрытие текущего модуля.

 

 

Предоставленный функционал является базовым и может быть легко расширен за счет добавления новых функциональных модулей.

 

Также во всех справочниках имеется флаг состояния записи

Флаг, который управляет отображением выбранной записи в других итоговых модулях.

Справочник типов продуктов

 

Справочник типов продуктов позволяет задавать типы продуктов, которые выпускает фабрика

 

 

Типы продуктов – это описание группы товаров, имеющие некоторое общее сходство. К примеру, белый и черный шоколад, шоколадный батончик имеет тип шоколад. А если имеется несколько видов шоколадных батончиков – то может быть создан отдельный тип «шоколадный батончик»

Справочник продуктов

 

Справочник продуктов позволяет управлять некоторым набором продуктом и группировать продукты по типам продуктов.

 

 

Для справочника продуктов доступен дополнительный модуль – Ингредиенты продукта, который позволяет задать ингредиенты, необходимые для производства указанного продукта.

 

Справочник ингредиентов

 

Ингредиенты – это составляющиеся продукта. Для каждого продукта необходимо некоторое количество ингредиентов для производства. Данный модуль позволяет оперировать списком доступных ингредиентов.

 

 

Количество ингредиентов может быть любым. Также список ингредиентов может быть доработан для задания дополнительных свойств вида «химическая формула» и т.п.

Справочник покупателей

 

Как правило, если что-то создается, то оно должно продаваться. Данный модуль позволяет составлять список потенциальных покупателей, которых может заинтересовать выпускаемая продукция.

 

 

Для справочника покупателей доступен дополнительный модуль – покупаемая продукция. Модуль позволяет задать выпускаемую продукцию потенциальному покупателю.

 

Справочник поставщиков

 

Любой создаваемый продукт имеет некоторые составные части. В случае с продуктами кондитерской фабрики – для создания продукции необходимы поставки ингредиентов. Данный модуль позволяет задать потенциальных поставщиков и поставки ингредиентов.

 

 

Поставки ингредиентов реализуются в дополнительном модуле – поставки ингредиентов. Модуль позволяет задать необходимый ингредиент потенциальному поставщику
